# crm-lead-automation
This project is an automated data pipeline that extracts structured data from web sources, processes it, and imports qualified leads into a CRM system via API.  The solution eliminates manual data collection and ensures reliable lead delivery directly into the sales funnel.
Features

Web scraping using Selenium

Structured data parsing and validation

Excel data transformation

Automated CRM API integration

Error handling and logging

Anti-blocking mechanisms (random delays, user-agent handling)

Tech Stack

Python

Selenium

Pandas

Requests

REST API integration

Architecture

Web Source → Scraper → Data Cleaning → Excel Transformation → CRM API → Sales Pipeline

Results

Reduced manual lead processing time by 80%+

Improved data consistency

Automated daily lead uploads
